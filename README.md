# ASCII-Art

## Introduction

* This is the C# version of Video2Text.
* It was assembled by _C#_, with the window of _WPF_.
* It can make asciis from a video, and you can view it through your command line.
* The project was originated from the Java version [Video2Text](https://github.com/Kiramei/Video2Text).

## Usage

1. Open ```Ascii Art.exe``` , and you'll see:

<img decoding="async" src="https://raw.githubusercontent.com/Kiramei/ASCII-Art/main/doc/show.png" width="60%"/>

2. If you want to open an existing asc file you've generated by the program, click ```Open``` and select it.
   
   If not, click ```Create``` . If the video is valid, you'll see:

<img decoding="async" src="https://raw.githubusercontent.com/Kiramei/ASCII-Art/main/doc/show1.png" width="60%"/>

  In this step, you can double click the Output directory box to change the output directory.

3. Click ```Start``` , and wait for the process.

<img decoding="async" src="https://raw.githubusercontent.com/Kiramei/ASCII-Art/main/doc/show2.png" width="60%"/>

If it has finished, you'll see:

<img decoding="async" src="https://raw.githubusercontent.com/Kiramei/ASCII-Art/main/doc/show3.png" width="60%"/>

4. Now You can click ```View``` to view the file at once!

<img decoding="async" src="https://raw.githubusercontent.com/Kiramei/ASCII-Art/main/doc/show4.png" width="60%"/>
